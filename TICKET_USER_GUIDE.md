## Troubleshooting

### Can't See the Download Button?
- Make sure you're logged in
- Refresh the page
- Check that you have active bookings

### Ticket Won't Open?
- Check your browser's pop-up blocker
- Try right-clicking the button and select "Open in new tab"
- Make sure you're connected to the internet

### Print Issues?
- Try saving as PDF first, then print the PDF
- Check your printer settings
- Ensure you have enough ink/toner
